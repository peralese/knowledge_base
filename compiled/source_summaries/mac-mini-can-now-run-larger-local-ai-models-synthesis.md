# Summary

Apple has officially approved TinyGPU drivers for AMD and NVIDIA GPUs on macOS, enabling Mac Mini users to run larger local AI models via eGPUs connected through Thunderbolt/USB4. This development enhances the computational capabilities of the Mac Mini, allowing it to host more powerful AI applications.

# Key Insights

- **Official Approval**: Apple has approved TinyGPU drivers for AMD and NVIDIA GPUs on macOS.
- **Enhanced Capabilities**: The Mac Mini can now run larger local AI models thanks to eGPUs connected via Thunderbolt/USB4.
- **Ease of Setup**: Installation instructions are available on TinyGPU's GitHub page, making it straightforward to set up the necessary hardware.
