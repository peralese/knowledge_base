# Summary

Apple has officially approved TinyGPU drivers for AMD and NVIDIA on macOS, enabling the Mac Mini to run larger local AI models through external GPUs (eGPUs) connected via Thunderbolt/USB4. The approval facilitates enhanced performance capabilities for users hosting OpenClaw agents or running complex AI models locally.

# Key Insights

- **Official Approval**: Apple has accepted TinyGPU drivers from the tiny corp, allowing AMD and NVIDIA GPUs to be used with macOS.
- **Enhanced Capabilities**: This development enables Mac Minis to handle larger local AI models, significantly boosting their computational power for tasks that require substantial GPU resources.
- **Ease of Setup**: Detailed installation instructions are available on TinyGPU's GitHub page, making it straightforward for users to set up and use the drivers.
