# Question

What are the main components of Karpathy's LLM Knowledge Base?

# Answer

The main components of Karpathy's LLM Knowledge Base are the `raw/`, `wiki/`, and `outputs/` directories, along with a schema file that guides how new material is processed and integrated.

### Supporting Details

- The `raw/` directory stores source documents that will be ingested into the knowledge base (Source: [[karpathy-s-llm-knowledge-base-build-an-ai-second-brain]]).
- The `wiki/` directory contains articles generated by an LLM based on the sources found in the `raw/` directory, ensuring a structured and consistent format (Source: [[karpathy-s-llm-knowledge-base-build-an-ai-second-brain]]).
- The `outputs/` directory holds responses to queries made against the knowledge base, maintaining a record of all interactions for future reference (Source: [[karpathy-s-llm-knowledge-base-build-an-ai-second-brain]).
- A schema file (e.g., `CLAUDE.md`) outlines how new material is processed and integrated into the system, as well as how queries are answered and linting passes conducted to maintain structural integrity and factual accuracy (Source: [[karpathy-s-llm-knowledge-base-build-an-ai-second-brain]).
